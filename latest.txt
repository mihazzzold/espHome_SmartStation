
📦 [0.1.1-beta] - 2026-03-15

📦# 🎉 Обновление для Home Assistant

📦# ✨ Новые возможности
  • **Native Update Entity**: В Home Assistant теперь виден статус обновления (Доступна новая версия).
  • **OTA Link**: Прямая ссылка на манифест обновления для автоматического обнаружения.

📦# 🛠️ Исправления
  • Улучшена обработка ошибок при проверке обновлений.
  • Оптимизирована структура метаданных для совместимости с ESPHome 2023.9+.

📦 [0.1.0-beta] - 2026-03-13

📦# 🎉 Первый релиз (Бета 0.1)

📦# ✨ Новые возможности
  • **espHome SmartStation**: Поддержка голосового ассистента, Hi-Fi звука (PCM5102A) и MEMS микрофона (INMP441).
  • **esp32c3-radar-light**: Интеграция радарного датчика присутствия и управления освещением.
  • **espHome HumidifierSystem**: Автоматизация системы увлажнения воздуха.
  • **OTA Updates**: Поддержка обновлений "по воздуху" через GitHub/OTA-server.
  • **Documentation**: Интерактивная документация на базе Astro/Starlight.

📦# 🛠️ Инструментарий
  • **generate_metadata.py**: Скрипт для автоматической генерации метаданных обновлений.
  • **Jinja2 шаблоны**: Все прошивки параметризированы для удобной настройки.
  • **Docker поддержка**: Готовые конфигурации для сборки документации и прошивок.
