
📦 [0.1.2-beta] - 2026-03-19

📦# 🎉 Обновление для Home Assistant (Новая версия)

  • **Internal Temperature Sensor**: Добавлена поддержка датчика внутренней температуры ESP32-C3.

  • Улучшена стабильность подключения к голосовому ассистенту.
  • Оптимизировано энергопотребление в режиме ожидания.

📦 [0.1.1-beta] - 2026-03-15

📦# 🎉 Обновление для Home Assistant

📦# ✨ Новые возможности
  • **Native Update Entity**: В Home Assistant теперь виден статус обновления (Доступна новая версия).
  • **OTA Link**: Прямая ссылка на манифест обновления для автоматического обнаружения.

📦# 🛠️ Исправления
  • Улучшена обработка ошибок при проверке обновлений.
  • Оптимизирована структура метаданных для совместимости с ESPHome 2023.9+.
